Chaos Sandbox

Interactive failure injection environment for OpenTelemetry exploration

Discord GitHub

What is this?

A sandbox environment for experimenting with chaos engineering and OpenTelemetry. Inject failures and observe how they affect your application's telemetry - no configuration needed.

What this is not

This is not a comprehensive OpenTelemetry demo. For that, check out the OpenTelemetry demo.

Recommended Audience

Developers exploring OpenTelemetry, those wanting quick experimentation without local setup, and anyone curious about observability in distributed systems.

What is a sandbox?

An isolated environment where you can conduct experiments independently without affecting other users. Each sandbox has its own telemetry stream.
